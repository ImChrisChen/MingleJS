# tips-text 内容提示组件



### 参数

| 参数名       | 说明                        | 默认值 |
| :----------- | --------------------------- | ------ |
| data-trigger | 触发事件 ‘click’ \| 'hover' | hover  |
| color        | 提示文本颜色                | ''     |
| data-label | 展示文本                    | ‘’     |



### 代码示例

```html
<button class="ant-btn ant-btn-primary"
        data-fn="tips-text"
        data-label="查看详情">
详情内容
</button>
```
