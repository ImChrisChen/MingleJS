# data-chart 图表组件



### 属性说明


| 参数         | 说明                                             | 默认值 |
| ------------ | ------------------------------------------------ | ------ |
| data-key     | 图表的x轴字段 按照下面的数据结构改值为： 'key'   | ''     |
| data-value   | 图表的Y轴字段 按照下面的数据结构改值为： 'value' | ''     |
| data-groupby | 图表的分组， 'key' \| 'value' \| ‘type’          |        |
|              |                                                  |        |
